# 🐧 SYSTEM-INFO v4.2

An advanced, native, and interactive system utility tool for Linux distributions (Ubuntu, Debian, Mint, Pop!_OS) developed by **Alexandru (balexandru15)**.

## 🚀 Features
* **Multilingual System:** Dynamic language selector with real-time interactive search (Supports English, Română, Español, Français, Deutsch, Italiano).
* **Smart System Cleanup:** Safely removes unused packages (`apt autoremove`) and clears old installer cache (`apt clean`) + frees unused RAM cache.
* **Detailed Resource Monitor:** Displays total, used, and free space for your Hard Drive (HDD) and Internal Memory (RAM).
* **Hardware Information:** Detects your exact PC hostname, current user, universal CPU model, and active Linux Kernel version.
* **Terminal Native Experience:** Runs cleanly on an alternative screen layout protecting your previous terminal command history.

## 📥 How to Install (One-Line Automated Setup)
To install this tool directly onto your system using native `apt`, just copy and paste the following command into your terminal:

```code:
curl -sL https://githubusercontent.com -o system-info.deb && sudo apt install ./system-info.deb && rm system-info.deb
```

Once the setup is complete, simply launch the utility by typing:
```code:
system-info
```
