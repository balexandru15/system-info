Instalare Rapidă din Terminal

Dacă vrei să instalezi direct ultima versiune (v4.6) folosind site-ul tău stabil, folosește această comandă:

curl -sL https://github.io -o system-info.deb && sudo apt install ./system-info.deb && rm system-info.deb && system-info
