Instalare Rapidă într-o Singură Linie

Copiază comanda de mai jos și lipește-o în terminalul tău pentru a instala direct ultima versiune:

curl -sL https://githubusercontent.com -o system-info.deb && sudo apt install ./system-info.deb && rm system-info.deb && system-info
